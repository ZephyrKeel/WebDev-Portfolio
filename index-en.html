Checkbox

Using the shift key to select multiple checkboxes

The goal of this project was to use JavaScript to make it so that clicking a second checkbox whilst holding down the shift key, would select all checkboxes between the first and second boxes that were clicked. This feature is so useful it has become mundane as it is present throughout many different types of applications that people use daily, such as an email inbox. Beyond the original goal of the #JS30 lesson, I also added a clear button to make the project feel a bit more complete. Click here to see it.

Video Player

Stylized HTML Video Player

In this #JS30 project the goal was to use HTML and CSS to create better looking controls for a video player, as compared to the stock controls of an HTML video player. Then, to use JavaScript to designate functions to the different controls and make them function harmoniously within the webpage. After creating the play/pause, volume, playback speed, and skip controls as the lesson directed, I also added and fullscreen control button. Additionally, I created functions that would display the current state of any slider while hovering the mouse over them, whether it be the current time of the video, the playback speed, or volume setting. Check it out.

Easter Eggs

Pop-ups that appear when a secret code is entered

This one is a #JS30 lesson that grew legs and went off to become something more. What was a simple lesson on Key Sequence Detection is now a fun little source of easter eggs on this webpage. It uses an EventListener on the window to detect keystrokes and saves the 8 most recent ones to an array. If the saved keystrokes match one of the secret keys I defined, an associated pop-up will appear with its unique easter egg. Try typing "gametime", "rickroll", or "ohexavem" to see the results.
